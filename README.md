# Clocks - 时钟

Display clocks for different time zones in the status bar

在状态栏显示不同时区的时钟

## Quick Start - 使用方法

After installing the extension, the current time will appear on the right side of the status bar

安装插件后, 在状态栏右侧即可显示始终

## Setting - 设置

- `hour12`: 12 hours - 12 小时制
- `showSecond`: Show second - 显示秒
- `worldClocks`: World clocks - 世界时钟
